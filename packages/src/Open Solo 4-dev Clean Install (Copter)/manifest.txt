Open Solo 4-DEV Clean Install (COPTER)
(Oct 2, 2019 / MCL)

This package contains Open Solo 4-DEV for testing!  This build includes ArduCopter 4-dev (master) as of 10/2/2019.  This package will install Open Solo 4-dev on the copter only.  You must also install the controller package.

If you have not read the instructions on the Open Solo github site, please do so before executing this installation!
http://www.github.com/opensolo/opensolo
